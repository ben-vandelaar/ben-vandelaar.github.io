Parentheses () - e.g. '1. Execute a function var result = doSomething()'
Parentheses are used in js for Grouping, containing expressions and parameters.
The Example probably doesn't do a good job of explaining what's happening.
When using the () in a function it's to be used as a paramater that will later be called when using the function.
e.g. var multiplyTwoNumbers = function(a,b){ result = a * b; here a and b are multiplied console.log(result); }

multiplyTwoNumbers(6,12) Here we add our own values to a and b to then be run through the function multiplying 6 by 12 and console logging 72

Parentheses are also used in a similar fasion when using Higher Order Functions such as "forEach" or
setTimeOut(function(){ console.log("Wake up!!")},50000);
