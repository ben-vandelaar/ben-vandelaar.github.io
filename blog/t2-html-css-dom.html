Q) What is meant by boxifying design?

A) Each Element is represented in HTML by a box. Your header is a box your content is a box and everything in that is a box like the Matryoshka dolls

What is the box model?

A)The Box Model consists of the margins, borders, padding and content, which together make up every HTML element on the page. Each of these aspects can be adjusted to create different styles, effects and spacing.
