Explain what functions are and why they are useful.
Functions are blocks of code that can be called upon (repeatedly) to perform a certain task or "function". It can take arguments, manipulate arguments/set variables and/or return a variable.
It helps make code easier to read and saves you from repeating yourself. They can also be kept in a seperate files to keep your file and script clean.
