Technical Blog QA

Q) What is grid based design?

A) Everything in HTML is a square, thus no matter how you look at it no matter what you're design will ensentaily be based on a grid like design.
This means understanding the grid layout can help speed up your design process and simplfy you code

Q) Whats all the hype about responsive webpages?

A) The hype? I don't think it's "hype". Hype to me implies some sort of whoohaa is involved. No. Responsive webpages are now a fact of modern life. Mobiles and Tablets have taken the lead in web usage compared to the home desktop in the last 2 years, not having a responsive well designed responsive sites for these mediums is essentially suicide.
