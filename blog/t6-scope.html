What is 'scope', and how it works in JavaScript.

Scope is the accessibility of variables, functions, and objects...
A quick example of this is from the w3schools.

// code here can NOT use carName

function myFunction() {
var carName = "Volvo";

// code here CAN use carName

}

It took me a little while to understand this one when I first started look at JavaScript. But the example above
does do a good job of explaining the rough idea of it.

When you create a variable on the Global scope it is accessible by all areas of your Javascript,
But when you create a variable inside a function or an object that "var" is only accessible inside that function..
In saying that it is also available to any functions inside that function.

Here's another example:

// Code in the global scope has access to variable `g`
// but not `o` nor `i`.

var g = 'global';

function outer () {
// Code in this function scope has access to variables
// `o` and `g`
// but not `i`.
var o = 'outer';

function inner () {
// Code in this function scope has access to variables
// `i`, `o` and `g`.

var i = 'inner';
}
}
