What a responsive site is, and why responsiveness is important.
To be a responsive website the website needs to be able to adapt and change to every type of screen, this is important as with new technologies people are viewing your websites on a wider variety of devices.
And if your website is not responsive and does't adapt itself to a persons device you have essentially lost yourself a customer.

What mobile first design is, and why it's important.
It's all about conveying information. Mobile first means to start designing website or app with the smallest possible screen in mind.
Designing like this will more clearly convey information and aid the customer with with having a more positive user experience when viewing your website on such a screen.
It's also good to note with all the restrictions with the size and interaction with the mobile anything you design will easily work on a desktop this can't be said the other way around.

What frameworks are, and their pros and cons.
Frame works are a pre-built css file that has a bunch of scripts designed to save you time when designing a website.
They are good for the basic layouts and help in easily making elements such as images responsive but lack in any real detail often requiring you to have a second css file for when you want to actually start customising the website.
They also lock you into a certain way of laying out your html and can sometimes break if not their rule set is not respected.
